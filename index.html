Proyectos
iFreelance
Sistema Administrativo de un Freelance

Es un proyecto desarrollado en Ruby on Rails, para el curso de Ingenieria de Software II, este proyecto consiste en menejar la parte administrativa de un Freelance, donde el usuario puede llevar control de sus proyectos, puede verificar el avance de estos mismos, ver el reporte de ingresos y egresos, ademas de generar reportes para manejo offline como facturas, gastos de representacion entre otros....

PC-UN
Sistema para el Manejo de Residuos PostConsumo

Es un proyecto desarrollado para el curso de Arquitectura del Software, este proyecto consiste en el manejo de residuos de Post Consumo, como Baterias, LLantas, Botellas, partes de Celulares, entre otros... Se manejan dos clientes uno Web y otro Movil, el objetivo es que el usuario una vez se conecte, pueda hacer un rastreo de que puntos mas cercanos tiene, para poder ir a depositar los residuos segun el tipo de residuo..
